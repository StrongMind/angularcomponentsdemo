Welcome to the StrongMind Component Demo!

The StrongMind Component Suite is an open source toolkit for developing with HTML, CSS, and Angular v2.0. Quickly integrate prebuilt StrongMind-styled components.

Requirements

Angular 2.0.0+ (https://angular.io)

UI Dependency

None

Installation

Basic Usage

Import the desired [components] in your `app.module` and add it to the declarations array. Alternatively, you can import desired [components] to a shared module to make it available across all modules in your Angular application.
